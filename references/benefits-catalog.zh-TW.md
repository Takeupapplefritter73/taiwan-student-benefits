## 1. GitHub Student Developer Pack

Pack 是所有福利的入口。請最優先申請。

**申請方式：** https://education.github.com/pack

## 12. 帳號綁定指南 — 信箱彈性

> **核心問題：** 我能用個人信箱（如 Gmail、公司信箱）當主帳號嗎？還是必須用 `.edu.tw` 註冊？
>
> 這很重要，因為 `.edu.tw` 信箱畢業後會失效。綁定在個人信箱的服務，畢業後仍能保留帳號和資料。

### C 類 — 透過 GitHub Pack（可用個人信箱）

這些服務透過你的 GitHub 帳號驗證。Pack 核准後，在合作夥伴網站用 GitHub 帳號登入兌換，**完全不需要在該網站提供 `.edu.tw` 信箱**。

| 服務 | 綁定方式 | 注意事項 |
|------|---------|---------|
| GitHub Copilot | 自動在你的 GitHub 帳號啟用 | — |
| GitHub Pro | 自動在你的 GitHub 帳號啟用 | — |
| DigitalOcean | 註冊時連結 GitHub 帳號 | 需信用卡 |
| Heroku | 透過 GitHub Education 入口兌換 | — |
| 1Password | 連結 GitHub 帳號 | 僅限個人方案（非家庭/團隊） |
| Frontend Masters | 透過 GitHub Education 入口兌換 | — |
| Educative | 透過 GitHub Education 入口兌換 | — |
| GitKraken | 透過 GitHub Education 入口兌換 | — |
| MongoDB Atlas | 從 GitHub Education 取得優惠碼 | 90 天內兌換；需信用卡 |
| Stripe | 透過 GitHub Education 入口兌換 | — |
| Name.com | 透過 GitHub Education 入口兌換 | 多種頂級域名可選 |

### A 類 — 可用個人信箱（另外驗證學生身份）

用你慣用的信箱註冊帳號。學生身份透過獨立流程驗證（SheerID、上傳學生證、或連結 GitHub Pack）。

| 服務 | 驗證方式 | 說明 |
|------|---------|------|
| JetBrains | GitHub Pack 連結 或 `.edu.tw` 或 ISIC 卡 | **最佳路徑：連結 GitHub Pack** — 完全不需 edu 信箱。2024 年 7 月起已移除文件上傳選項。 |
| Autodesk | SheerID（上傳學生證或 edu 信箱） | 帳號信箱可以是個人信箱 |
| Spotify 學生方案 | SheerID | 台灣的大學有支援。可用任何信箱註冊。 |
| YouTube Premium | SheerID | 綁定你的 Google 帳號；驗證是獨立流程 |
| Perplexity Pro | SheerID | 帳號信箱和驗證信箱可以不同 |
| Miro Education | edu 信箱或上傳學生證 | 若 edu 信箱未被辨識，可聯繫客服並附學生證 |
| AWS Educate | 偏好 edu 信箱，個人信箱需補件 | 用個人信箱需回覆附上學生證；審核較慢 |
| Postman Student Expert | 完成官方訓練模組 | 不需任何信箱驗證；任何人都能取得徽章 |

### B 類 — 必須用 `.edu.tw` 信箱

這些服務**要求**以 `.edu.tw` 信箱作為登入/帳號信箱，無法繞過。

| 服務 | 為何需要 edu | 建議 |
|------|------------|------|
| Cursor Pro | 帳號信箱必須與 SheerID 驗證信箱完全一致 | 信箱不符是被拒的首要原因。台灣透過 SheerID 驗證可能需要人工審核。 |
| Notion Plus | 僅辨識 edu 信箱域名自動升級 | 無文件上傳選項。學校需在 WHED 資料庫中。 |
| MS 365 Education | 授權綁定學校信箱域名 | 到學校資訊中心確認是否有提供 |
| Azure for Students | 需要教育機構信箱驗證 | **可能的替代方案：** 有用戶回報透過 GitHub Pack 連結可繞過 edu 信箱要求 |
| Apple Music 學生方案 | UNiDAYS 驗證需要 edu 信箱 | UNiDAYS 支援台灣（114 國） |
| Figma Education | 強烈偏好 edu 信箱；個人信箱常被拒 | 被拒絕的話，寄信至 education@figma.com 附學生證申請人工審核 |

### D 類 — 透過學校入口

| 服務 | 運作方式 |
|------|---------|
| Google Cloud $300 | 教授申請額度後發送 coupon 給學生。**Coupon 可套用到任何 Google 帳號。** |
| Zoom Education | 學校統一授權；洽詢學校資訊中心 |
| Canva Education | 需要學校訂閱「Campus」方案；沒有個人學生方案 |

### 關鍵洞見

1. **SheerID 是通用的學生驗證平台** — Spotify、YouTube、Autodesk、Perplexity、Cursor 等服務都使用它。涵蓋 191 個國家、超過 20 萬所學校。台灣的公立大學（包括勤益科大、台大、陽明交大、成大）大多在資料庫中。

2. **GitHub Pack 是最強大的捷徑** — 11 項服務不需任何額外信箱驗證即可兌換。先申請 Pack，再透過它一次領取。

3. **JetBrains 最佳路徑 = GitHub Pack** — 有了核准的 Pack，JetBrains 完全不需你的 edu 信箱。只要在申請時連結 GitHub 帳號即可。

4. **Namecheap nc.me 有地區限制** — 主要支援美國/英國/加拿大/澳洲的 `.edu` 信箱。台灣的 `.edu.tw` 可能不符資格。改用 **Name.com**（透過 GitHub Pack）取得免費網域。

5. **B 類服務建議設定信箱轉寄** — 被迫使用 `.edu.tw` 的服務，請設定學校信箱轉寄到個人信箱，避免畢業後漏接續約通知。

6. **Azure 替代方案值得嘗試** — 有用戶成功透過 GitHub Student Pack 連結來啟用 Azure for Students，繞過 edu 信箱驗證。結果因人而異。
